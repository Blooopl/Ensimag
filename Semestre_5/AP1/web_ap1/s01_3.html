Algo-Prog 1 : Chapitre 1 : Séance 01 : Mise en forme

La forme, c’est le fond qui remonte à la surface

Vous trouvez ça lisible ?

Le programme suivant est un programme Go correct :

package main
           import "fmt"; func main() {
fmt.Println(         "Bonjour je suis très mal formaté non ?") }

Tapez dans votre terminal la commande cat > mal_formate.go puis entrée pour l’exécuter. cat attend son entrée sur l’entrée standard : copiez-collez dans le terminal le code ci-dessus en terminant votre saisie par un ctrl-d (cela signifie qu’il faut appuyer en même temps sur les touches ctrl et d).

En exécutant la commande ls -hl dans votre terminal vous voyez que la commande cat > mal_formate.go a bien créé le fichier mal_formate.go contenant le code que nous avons copié-collé.

Lancez maintenant la commande go run mal_formate.go et vous devriez bien voir l’affichage prévu : ce programme fonctionne.

Formatage automatique

Bien que correct, ce programme est inacceptable et vaudrait sûrement une très mauvaise note à l’élève qui le rendrait tel-quel, car le formatage rend ce programme parfaitement illisible. Pour corriger ce problème, le langage Go fournit l’outil go fmt dont l’objectif est de formater automatiquement notre code.

Tapez dans votre terminal la commande cp mal_formate.go bien_formate.go pour créer une copie du fichier mal-formaté. Vous pouvez vous convaincre qu’il contient bien exactement la même chose en l’affichant avec la commande cat bien_formate.go.

Exécutez maintenant la commande go fmt bien_formate.go, puis tapez à nouveau cat bien_formate.go : on voit que le code a été formaté tout seul correctement !

Maintenant, créez un fichier vide avec la commande touch aussi_bien_formate.go puis ouvrez-le avec VSCode en tapant code aussi_bien_formate.go & dans le terminal. Copiez le code mal-formaté du début et collez-le dans VSCode, puis sauvegardez en tapant ctrl-s.

On voit que VSCode appelle tout seul go fmt dès qu’on sauvegarde un fichier .go, ce qui va beaucoup nous aider ce semestre !

Enfin, tapez dans un terminal la commande go fais_mon_tp_a_ma_place : non, malheureusement, ça ne marche pas…
