Factorisation de code

Dans cet exercice à l’énoncé très court, vous devez utiliser des boucles pour factoriser le code ci-dessus, dont le but est de dessiner un échiquier en SVG :

Vous aurez besoin pour cela du module SVG fourni dans l’exercice sur la programmation modulaire de la séance 2 : vous pouvez utiliser un lien symbolique pour le rendre accessible dans le répertoire courant (commande ln -s).

Le code à factoriser est le suivant :

package main

import "fmt"

const coteCase = 25.0

const nbrCases = 4

func echiquier() {
    fmt.Println(debutImageSVG(coteCase*nbrCases, coteCase*nbrCases))
    // Ligne 1
    fmt.Println(rectangleSVG(pointSVG{0, 0}, coteCase, coteCase, "black"))
    fmt.Println(rectangleSVG(pointSVG{coteCase, 0}, coteCase, coteCase, "white"))
    fmt.Println(rectangleSVG(pointSVG{coteCase * 2, 0}, coteCase, coteCase, "black"))
    fmt.Println(rectangleSVG(pointSVG{coteCase * 3, 0}, coteCase, coteCase, "white"))
    // Ligne 2
    fmt.Println(rectangleSVG(pointSVG{0, coteCase}, coteCase, coteCase, "white"))
    fmt.Println(rectangleSVG(pointSVG{coteCase, coteCase}, coteCase, coteCase, "black"))
    fmt.Println(rectangleSVG(pointSVG{coteCase * 2, coteCase}, coteCase, coteCase, "white"))
    fmt.Println(rectangleSVG(pointSVG{coteCase * 3, coteCase}, coteCase, coteCase, "black"))
    // Ligne 3
    fmt.Println(rectangleSVG(pointSVG{0, coteCase * 2}, coteCase, coteCase, "black"))
    fmt.Println(rectangleSVG(pointSVG{coteCase, coteCase * 2}, coteCase, coteCase, "white"))
    fmt.Println(rectangleSVG(pointSVG{coteCase * 2, coteCase * 2}, coteCase, coteCase, "black"))
    fmt.Println(rectangleSVG(pointSVG{coteCase * 3, coteCase * 2}, coteCase, coteCase, "white"))
    // Ligne 4
    fmt.Println(rectangleSVG(pointSVG{0, coteCase * 3}, coteCase, coteCase, "white"))
    fmt.Println(rectangleSVG(pointSVG{coteCase, coteCase * 3}, coteCase, coteCase, "black"))
    fmt.Println(rectangleSVG(pointSVG{coteCase * 2, coteCase * 3}, coteCase, coteCase, "white"))
    fmt.Println(rectangleSVG(pointSVG{coteCase * 3, coteCase * 3}, coteCase, coteCase, "black"))
    fmt.Println(finImageSVG())
}

func main() {
    echiquier()
}
