Algo-Prog 1 : Chapitre 3 : Séance 19 : Ensembles

✓ Correction : s19_2.go

Énoncé

Dans cet exercice, nous allons manipuler des ensembles d’entiers. Go ne fournit pas de type ensemble, mais il est très simple d’en construire en utilisant une map dont les valeurs sont des booléens : type ensembleEntiers map[int]bool.

Vous devez implanter les fonctions classiques ci-dessous.

créerEnsemble() ensembleEntiers alloue et renvoie un nouvel ensemble d’entiers.

insérer(val int, ens ensembleEntiers) insère la valeur dans l’ensemble.

supprimer(val int, ens ensembleEntiers) supprime la valeur de l’ensemble : la fonction prédéfinie delete (documentation) vous sera vraisemblablement utile.

afficher(ens ensembleEntiers, nom string) affiche les valeurs de l’ensemble dans l’ordre croissant et préfixé par le nom passé en paramètre : par exemple, l’appel afficher(ensA, "A") pourra produire l’affichage A = { 1 2 3 4 }.

Si vous travaillez sur votre machine perso avec au moins la version 1.23 de Go (go version pour savoir), vous pouvez utiliser les fonctions maps.Keys (documentation) et slices.Sorted (documentation) pour produire un affichage ordonné.

Si vous travaillez sur les machines de l’école qui sont en 1.22, vous devez :

  • créer un slice lesClés avec make([]int, 0)
  • le remplir en itérant sur ens avec un for clé := range ens qui fait un append à chaque tour de boucle pour ajouter la clé courante dans le slice ;
  • utiliser slices.Sort pour trier le slice ;
  • et enfin itérer sur les éléments du slice pour afficher chaque clé dans le bon ordre.

appartient(val int, ens ensembleEntiers) bool teste si la valeur appartient à l’ensemble ; on rappelle qu’une map renvoie la valeur nulle du type de ses valeurs (c’est-à-dire false dans le cas de booléens) si la clé utilisée est absente.

union(ens1, ens2 ensembleEntiers) ensembleEntiers renvoie un nouvel ensemble égal à l’union des deux ensembles passés en paramètre ; la fonction maps.Copy (documentation) devrait faciliter le travail.

intersection(ens1, ens2 ensembleEntiers) ensembleEntiers renvoie un nouvel ensemble égal à l’intersection des deux ensembles passés en paramètre.

différence(ens1, ens2 ensembleEntiers) ensembleEntiers renvoie un nouvel ensemble égal à la différence des deux ensembles passés en paramètre.

différenceSymétrique(ens1, ens2 ensembleEntiers) ensembleEntiers renvoie un nouvel ensemble égal à la différence symétrique (c’est-à-dire les valeurs présentes dans un des deux ensembles, mais pas dans les deux) des deux ensembles passés en paramètre.

Enfin, vous implanterez une fonction main pour tester vos fonctions, produisant par exemple un affichage comme ci-dessous :

A = { 1 2 3 4 }
B = { 3 4 5 6 }
A | B = { 1 2 3 4 5 6 }
A & B = { 3 4 }
A - B = { 1 2 }
B - A = { 5 6 }
A ^ B = { 1 2 5 6 }
